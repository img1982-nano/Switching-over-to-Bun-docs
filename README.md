# Switching-over-to-Bun-docs
This is how to migrate to the "bun" package manager.
# How to migrate to "bun"
1. Delete ```pnpm-lock.yaml```,```package-lock.json```,and more
2. Run ```bun update --latest```
3. Edit ```package.json```
   - ```npm ...``` to ```bun ...```
4. (Option) Commit(& Push) to Github
